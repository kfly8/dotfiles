# Setup

## [Homebrew](https://brew.sh)

```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew bundle
```

## Create dotfiles link

```sh
git clone https://github.com/kfly8/dotfiles.git
cd dotfiles
./create-link
```
