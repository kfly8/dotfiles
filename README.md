# Installation 


## clone and setup

    ```
    cd ~
    git clone https://github.com/kfly8/dotfiles.git 
    cd dotfiles 
    sh setup.sh
    ```

## and make [vimproc](https://github.com/Shougo/vimproc.vim).

    ```
    cd vim/neobundle/vimproc
    make
    ```
