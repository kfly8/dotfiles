kenta's dotfiles
--------

# Installation 


<pre>
cd ~
git clone https://github.com/kfly8/dotfiles.git 
cd dotfiles 
sh setup.sh
</pre>

and install [vundle](https://github.com/gmarik/vundle)
<pre>
git clone https://github.com/gmarik/vundle ~/.vim/vundle.git
</pre>
